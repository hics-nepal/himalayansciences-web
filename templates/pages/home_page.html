{% if request.LANGUAGE_CODE == 'ne' %}तथ्याङ्कको अभाव{% else %}The gap in the data{% endif %}

{% if request.LANGUAGE_CODE == 'ne' %} नेपाल हाम्रो समयका केही सबैभन्दा महत्त्वपूर्ण वैज्ञानिक प्रश्नहरूको प्रतिच्छेदनमा अवस्थित छ — वायुमण्डलीय प्रदूषण, भूकम्पीय जोखिम, जलवायु अनुकूलन, कस्मिक रे भौतिकशास्त्र। यो पृथ्वीमा सबैभन्दा कम उपकरण जडान गरिएका देशहरूमध्ये एक पनि हो। {% else %} Nepal sits at the intersection of some of the most important scientific questions of our time — atmospheric pollution, seismic hazard, climate adaptation, cosmic ray physics. It is also one of the most under-instrumented countries on Earth. {% endif %}

{% if request.LANGUAGE_CODE == 'ne' %}यसलाई परिवर्तन गर्न HICS ले मापन पूर्वाधार निर्माण गरिरहेको छ।{% else %}HICS is building the measurement infrastructure to change this.{% endif %}

{% if request.LANGUAGE_CODE == 'ne' %}वायुमण्डलीय अनुगमन{% else %}Atmospheric monitoring{% endif %}
{% if request.LANGUAGE_CODE == 'ne' %}नेपालको सम्पूर्ण उचाइ ढाँचा — तराईदेखि उच्च हिमालसम्मको वायुमण्डल — लगभग पूर्ण रूपमा मापन विहीन छ। इन्डो-ग्यान्जेटिक समथर भूभागबाट आउने प्रदूषण हिमालसम्म पुग्छ — तर त्यहाँ के पुग्छ, वा उचाइसँगै कसरी परिवर्तन हुन्छ भन्ने बारेमा लगभग कुनै तथ्याङ्क छैन।{% else %}Nepal's atmosphere across its full altitude gradient — from the Terai to the high Himalaya — is almost entirely unmeasured. Transboundary pollution from the Indo-Gangetic Plain reaches the mountains — there is almost no data on what arrives, or how it changes with altitude.{% endif %}
{% if request.LANGUAGE_CODE == 'ne' %}कस्मिक रे फ्लक्स{% else %}Cosmic ray flux{% endif %}
{% if request.LANGUAGE_CODE == 'ne' %}नेपालको भौगोलिक उचाइको ढाँचा — ६० मिटरदेखि ८,८४८ मिटरसम्म — पृथ्वीमै सबैभन्दा नाटकीय छ। तर यसलाई कस्मिक रे म्योन फ्लक्सको लागि कहिल्यै पनि व्यवस्थित रूपमा मापन गरिएको छैन। HICS को म्योन डिटेक्टरले यो तथ्याङ्क सेट तयार गर्नेछ।{% else %}Nepal's altitude gradient — from 60m to 8,848m — is the most dramatic on Earth. It has never been systematically measured for cosmic ray muon flux. HICS's muon detector will generate this dataset.{% endif %}
